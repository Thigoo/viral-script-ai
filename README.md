Viral Script is an AI-powered storytelling agent designed to create and refine high-retention scripts for short-form videos (Reels, Shorts, TikTok).

Built using LangChain, LangGraph, and LLM orchestration, the project explores multi-step generation pipelines (generate → critique → refine) to improve content quality and engagement.

This project serves as:

- A technical lab for learning LLM architecture
- A product MVP for content creators
- A case study in applied AI systems design
